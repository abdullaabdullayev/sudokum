A sleek, modern, and interactive Sudoku game built with **Java (Spring Boot)** and **Vanilla JavaScript**. This project features a dual-input system, symmetric board generation, and a responsive UI.

---

## ✨ Features

* **Dual-Input Mode:** * *Mode A:* Select a tile first, then click a number.
    * *Mode B:* Select a number first, then fill multiple tiles rapidly.
* **Smart Board Generation:** Symmetric tile removal for balanced and solvable puzzles.
* **Modern UI/UX:** * Soft-UI design with smooth transitions.
    * Automatic selection clearing (Escape/Click outside).
    * No annoying text cursors in tiles.
* **Validation:** Real-time logic check for rows, columns, and 3x3 grids.
* **Responsive Backend:** Spring Boot REST API for generating and validating puzzles.
